# ロボッチャにおける自動投球システムにおける画像解析パートのサンプル

## 概要
ロボッチャと呼ばれるLEGOロボットを用いた競技において、デモンストレーションとしての自動投球ロボットの開発を行いました。システムとしてはカメラでフィールドの状態を撮影し、ボールの位置情報を抽出、その後UIに反映します。そして任意の指定を指定すればロボットが自動で投球を行うというものでした。
その際に担当していた画像解析部分について、サンプルコードを公開します。

## 主な機能
- **[機能1]**: cv2.VideoCapture()による撮影
- **[機能2]**: カラーマスクとcv2.bitwise_and()によるカラーボールの抽出
- **[機能3]**: cv2.morphologyEx()によるノイズ抑制
- **[機能4]**: cv2.HoughCircles()によるボールの発見

## 使用技術
- **Language**: Python 3.12
- **Framework/Library**: OpenCV
